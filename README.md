# Finance-Management

Finance Management Application
A comprehensive Personal Finance Management Application developed with Spring Boot for the backend and React for the frontend. The app provides an easy-to-use platform for users to track their financial activities, including income, expenses, savings, and investments.

Key Features:
User Authentication: Secure login and registration implemented using JWT for authentication and authorization.
Dashboard: Visual representation of financial data with Chart.js, providing insights into monthly income, expenses, and savings trends.
Expense Tracking: Categorize and track daily expenses, helping users manage their budget efficiently.
Income Management: Record and monitor various sources of income.
Savings Goals: Set and track savings goals, ensuring better financial planning.
Investment Overview: Keep track of investments and performance over time.
Responsive Design: Built with React, offering a smooth and responsive user experience across devices.
Technology Stack:
Backend: Spring Boot (Java), PostgreSQL
Frontend: React.js, Chart.js
Authentication: JWT (JSON Web Tokens)
API Integration: RESTful API between frontend and backend
This project aims to provide a scalable and user-friendly solution for personal financial management.
